Assistant

public class Assistant

The IBM Watson™ Assistant service combines machine learning, natural language understanding, and an integrated dialog editor to create conversation flows between your apps and your users.

  • init(version:)

    Create a Assistant object.

    This initializer will retrieve credentials from the environment or a local credentials file. The credentials file can be downloaded from your service instance on IBM Cloud as ibm-credentials.env. Make sure to add the credentials file to your project so that it can be loaded at runtime.

    If credentials are not available in the environment or a local credentials file, initialization will fail. In that case, try another initializer that directly passes in the credentials.
